# robinbril.com

Personal website and AI consulting platform for **Robin Bril** — Senior AI Engineer specializing in AI agents, multi-agent systems, and process automation for businesses.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D**: Three.js (point cloud hero)
- **Deployment**: Vercel
- **Domain**: [robinbril.com](https://robinbril.com)

## Key Features

- Dutch + English localization
- SEO-optimized service landing pages
- Schema.org structured data (Person, Organization, Service, FAQ)
- AI chat widget
- Voice assistant demo pages
- Blog with markdown rendering

## Development

```bash
npm install
npm run dev
```

Available at [http://localhost:3000](http://localhost:3000).

## Structure

- `/src/app` — Next.js app router pages and layouts
- `/src/components` — React components (sections, UI, SEO)
- `/src/lib` — Configuration, schemas, i18n, metadata utilities
- `/public` — Static assets and images
